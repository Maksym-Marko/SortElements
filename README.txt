SortElements – скрипт для выбора каждого первого, второго, третьего, четвертого, пятого элемента с шагом в пять элементов.

ОПИСАНИЕ:
Иногда возникает необходимость выбрать элементы в наборе, например, каждый пятый. Эту функцию и проделывает данный скрипт.

НАСТРОЙКА:
var step = 5, // шаг (через какое количество элементов нужно выбрать определенный)
			block1 = 0, // элемент «block1» - первый элемент
			block2 = 1,  // элемент «block2» - второй  элемент
			block3 = 2, 
			block4 = 3,
			block5 = 4;

Далее вручную прописываем код для каждого элемента:
if( indexBlock == block1 ){

	// …

	block1 += step; // Сделать шаг

}
